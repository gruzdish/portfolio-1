== Description ==

Description: Vikata is a WPKoi theme under GPL license. You can find theme demos at wpkoi.com. Vikata has a black and yellow, powerful style and design for portfolio, business or agency content. The theme is multipurpose, so if You like the style, You can use it to represent Your art, business, an event, show Your portfolio, start a blog, start a webshop and sell Your products etc. The demos use Elementor, WooCommerce and Contact Form 7 plugins. The premium demo uses the Vikata premium plugin also. Vikata has an easy to use admin with a lot of customizer functions, so You can build Your dream without coding knowledge. The theme is responsive so looks good on almost every devices. You can setup multiple header styles, navigation styles and layouts. Build You site with Vikata Theme and share it with the world! You can also find a documentation with an install guide and a tutorial for the screenshot look on wpkoi.com/docs.

= Installation =

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Vikata in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
